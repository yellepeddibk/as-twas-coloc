% ── Introduction ──────────────────────────────────────────────
\section{Introduction}

Ankylosing spondylitis (AS) is a seronegative inflammatory arthritis
predominantly affecting the axial skeleton, with a population prevalence of
approximately 0.1--0.5\% in European populations \citep{Reveille2011}.
The disease exhibits high heritability (estimated 70--90\%), with HLA-B*27
as the largest genetic risk factor, accounting for $\sim$20--30\% of
genetic variance \citep{Brown2010}.
Genome-wide association studies (GWAS) have identified over 100 risk loci
beyond the MHC region, yet the majority of variants lie in non-coding regions,
complicating interpretation of their molecular mechanisms \citep{Ellinghaus2016}.

Transcriptome-wide association studies (TWAS) offer a complementary approach
to GWAS by integrating expression quantitative trait loci (eQTLs) with
GWAS summary statistics to identify genes whose predicted expression is
associated with disease risk \citep{Gamazon2015,Barbeira2018}.
S-PrediXcan (MetaXcan) implements a summary-statistic-based approach that
enables multi-tissue TWAS without requiring individual-level genotype data
\citep{Barbeira2018}.
The GTEx v8 resource provides high-quality eQTL prediction models trained
on 49 tissues from 838 post-mortem donors, enabling tissue-specific
prioritization of disease genes \citep{GTExConsortium2020}.

A critical limitation of TWAS is that statistical associations may arise from
coincidental linkage disequilibrium rather than a shared underlying causal
variant.
Bayesian colocalization using coloc.abf \citep{Giambartolomei2014} formally
tests whether the GWAS signal and the eQTL signal at a given locus are driven
by the same variant (posterior probability of hypothesis H4; PP4).
High PP4 (typically $\geq 0.7$) provides strong evidence that the disease
GWAS signal and the tissue-specific eQTL signal co-localize, supporting the
gene as a causal driver of AS risk in that tissue.

Here, we describe a reproducible pipeline integrating GWAS summary statistics
for AS with GTEx v8 S-PrediXcan TWAS followed by coloc-based prioritization,
with the aim of identifying tissue-specific candidate drug targets for AS.
The pipeline is designed for transparency, modularity, and extensibility to
future AS GWAS or eQTL data.

\subsection{GWAS Harmonization}

Raw summary statistics were harmonized to hg38 using the following steps:
\begin{enumerate}
  \item Column names were standardized to a canonical schema
        (varID, chrom, pos, effect\_allele, non\_effect\_allele, beta, se, pvalue, N).
  \item Where applicable, coordinates were lifted from hg19 to hg38 using
        \placeholder{liftover tool (e.g., pyliftover, CrossMap)}.
  \item GTEx-style variant identifiers (varIDs) were constructed as
        \texttt{chr\{chrom\}\_\{pos\}\_\{ref\}\_\{alt\}\_b38}.
        This format is required for matching GWAS variants to GTEx
        PredictDB prediction models.
  \item Ambiguous palindromic single-nucleotide polymorphisms (AT/CG) were
        removed by default to prevent misalignment of effect directions across
        strands, as these cannot be reliably strand-disambiguated without
        allele frequency information.
  \item Variants with missing effect sizes, standard errors, or p-values were
        excluded.
\end{enumerate}

\subsection{Transcriptome-Wide Association Study}

TWAS was performed using S-PrediXcan (MetaXcan v\placeholder{version})
\citep{Barbeira2018} with GTEx v8 mashr-based prediction models
\citep{GTExConsortium2020,Barbeira2019}.
\placeholder{N} tissues relevant to AS pathophysiology were analysed,
including \placeholder{tissue list, e.g., Whole Blood, Colon Sigmoid,
Small Intestine Terminal Ileum, Spleen, Muscle Skeletal, Lung}.

To control for multiple testing, Benjamini-Hochberg false-discovery rate
(FDR) correction was applied globally across all gene-tissue pairs tested.
Gene-tissue pairs with FDR-adjusted q-value $< 0.05$ were retained as
TWAS-significant hits.

\subsection{Bayesian Colocalization}

For each TWAS-significant gene-tissue pair, we performed Bayesian
colocalization using the coloc.abf function from the \texttt{coloc} R
package (v\placeholder{version}) \citep{Giambartolomei2014}.
Regional GWAS summary statistics ($\pm$1\,Mb around the gene transcription
start site) and tissue-matched GTEx v8 eQTL summary statistics were extracted.
All SNPs within the region were included without pre-filtering by p-value,
as removing non-significant SNPs inflates PP4 estimates \citep{Wallace2021}.

Effect sizes (beta) and their variances ($\sigma^2 = \mathrm{SE}^2$;
\textit{varbeta}) were provided as inputs to coloc.abf.
Standard coloc priors were used: $p_1 = 10^{-4}$, $p_2 = 10^{-4}$,
$p_{12} = 10^{-5}$.

Coloc results were interpreted using three complementary metrics:
\begin{itemize}
  \item PP4 $\geq 0.7$: primary evidence for a shared causal variant.
  \item PP3 + PP4 $\geq 0.8$: powered locus with sufficient signal to
        distinguish hypotheses H3 and H4.
  \item PP4 / (PP3 + PP4) $\geq 0.9$: strong preference for shared
        signal over distinct signals.
\end{itemize}

For loci with potential multiple independent signals
(\placeholder{if applicable}), coloc.susie was applied as a sensitivity
analysis \citep{Wallace2021}.

\subsection{Reproducibility}

All analyses were performed using a custom Python pipeline
(available at \url{https://github.com/yellepeddibk/as-twas-coloc}).
Each run saves a provenance manifest containing the pipeline version,
git commit hash, configuration snapshot, coloc priors, and input file checksums.
Random seeds were fixed throughout.
The pipeline runs in a lightweight mock mode for CI/CD validation without
requiring external data.

% ── Limitations ──────────────────────────────────────────────
\section{Limitations}

The current analysis has several limitations that should be considered when
interpreting findings.
First, the GTEx v8 eQTL models are derived from post-mortem tissues, which
may not fully capture in vivo inflammatory states relevant to AS.
Second, the application of coloc.abf assumes a single causal variant per locus;
for loci with multiple independent signals, coloc.susie should be applied.
Third, the GTEx sample sizes for individual tissues may be insufficient to
detect eQTLs for all disease-relevant genes.
